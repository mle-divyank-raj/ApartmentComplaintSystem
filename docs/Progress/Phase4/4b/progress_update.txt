## Architecture Rules Applied

- Zero business logic in ViewModels — only repository delegation
- Sealed UiState (Idle/Loading/Success/Error) per screen
- StateFlow only (no LiveData, no sharedFlow for UI)
- collectAsStateWithLifecycle() in all @Composable screens
- Domain interfaces in domain/repository/, implementations in data/repository/
- DTOs carry @SerializedName, domain models are clean Kotlin data classes
- Hilt @HiltViewModel + @Inject constructor everywhere
- BuildConfig.API_BASE_URL from local.properties (gitignored)
- Bearer JWT interceptor in OkHttpClient
